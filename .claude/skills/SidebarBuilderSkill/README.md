# Sidebar Builder Skill

## Purpose
The Sidebar Builder Skill creates and maintains navigation structures for the Physical AI & Humanoid Robotics course book.

## Capabilities
- Generate Docusaurus sidebar configurations
- Create logical navigation hierarchies
- Organize content by modules and sections
- Ensure consistent navigation structure
- Update navigation as content is added
- Validate navigation links and structure

## Usage Guidelines
- Organize content in logical hierarchical order
- Group related content under appropriate headings
- Maintain consistent naming conventions
- Ensure all content is accessible through navigation
- Keep navigation structure aligned with course modules
- Update navigation when new content is added

## Standards
- Follow Docusaurus sidebar configuration conventions
- Maintain consistent ordering throughout the course
- Use descriptive labels that match content titles
- Group content at appropriate levels of granularity
- Ensure navigation reflects the logical course progression
- Test navigation links for accuracy and functionality
