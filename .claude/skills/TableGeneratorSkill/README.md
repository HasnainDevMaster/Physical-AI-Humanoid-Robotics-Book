# Table Generator Skill

## Purpose
The Table Generator Skill creates responsive, well-formatted tables for the Physical AI & Humanoid Robotics course book.

## Capabilities
- Generate Markdown tables with proper formatting
- Create responsive tables that work on all devices
- Format comparison tables for different concepts or technologies
- Create data tables for specifications and parameters
- Generate tables with appropriate headers and alignment
- Ensure tables are accessible and readable

## Usage Guidelines
- Use appropriate column alignment based on content type
- Include descriptive headers for all columns
- Keep tables concise and focused on one concept
- Use tables for comparison of related items
- Ensure tables are responsive on mobile devices
- Include table captions when necessary for clarity

## Standards
- Follow Markdown table syntax conventions
- Use consistent formatting throughout the course
- Ensure tables enhance rather than duplicate textual content
- Apply appropriate styling through Docusaurus features
- Test table responsiveness across device sizes
- Use tables to present data that would be difficult to express in text
