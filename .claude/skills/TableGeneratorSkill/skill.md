# TableGeneratorSkill

## Purpose
The TableGeneratorSkill specializes in creating responsive and well-formatted tables for the Physical AI & Humanoid Robotics course documentation.

## Capabilities
- Generate Markdown tables with proper formatting
- Create responsive tables that work on all devices
- Format complex data in tabular format
- Generate comparison tables for technical concepts
- Create specification and parameter tables

## Usage Context
- Creating comparison tables for different technologies
- Formatting technical specifications
- Organizing complex information in tabular format
- Creating parameter and configuration tables
- Generating data comparison tables

## Guidelines
- Use Markdown table syntax with proper alignment
- Keep tables concise and readable
- Use appropriate headers for clarity
- Ensure tables are responsive on mobile devices
- Include only essential information in tables
- Use tables to enhance understanding, not complicate it
