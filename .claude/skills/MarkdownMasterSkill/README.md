# Markdown Master Skill

## Purpose
The Markdown Master Skill ensures consistent, well-formatted Markdown content across the Physical AI & Humanoid Robotics course book.

## Capabilities
- Apply consistent formatting to Markdown documents
- Ensure proper heading hierarchy (H1, H2, H3, etc.)
- Format code blocks with appropriate language tags
- Create properly structured lists and tables
- Apply Docusaurus-specific Markdown features
- Ensure accessibility compliance in Markdown

## Usage Guidelines
- Maintain consistent heading levels throughout documents
- Use appropriate code block language identifiers
- Apply proper emphasis and formatting
- Ensure all images have appropriate alt text
- Follow accessibility best practices
- Use Docusaurus-specific Markdown extensions where appropriate

## Standards
- Follow the course's style guide for terminology
- Maintain consistent voice and tone
- Use appropriate cross-references between modules
- Apply proper metadata in frontmatter where needed
- Ensure all content is properly structured for navigation
