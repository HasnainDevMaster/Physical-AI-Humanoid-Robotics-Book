# MarkdownMasterSkill

## Purpose
The MarkdownMasterSkill provides expert knowledge and assistance for creating and formatting Markdown content for the Physical AI & Humanoid Robotics course documentation.

## Capabilities
- Generate well-formatted Markdown content
- Apply consistent formatting and styling
- Create properly structured documentation
- Follow course-specific Markdown guidelines
- Generate tables, lists, and other Markdown elements

## Usage Context
- Creating course content in Markdown format
- Formatting technical documentation
- Creating consistent styling across course materials
- Converting content to appropriate Markdown format

## Guidelines
- Follow Docusaurus Markdown conventions
- Use appropriate heading levels (h1 for main titles, h2-h4 for sections)
- Include proper frontmatter with sidebar_position
- Use appropriate code blocks with language specification
- Include relevant links and cross-references
- Maintain consistent formatting across all course materials
